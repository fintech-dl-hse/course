# Вопросы для квиза по ноутбуку семинара

Примени команду к **открытому ноутбуку** семинара (`seminars/**/*_seminar_*.ipynb`). Сгенерируй **10 вопросов** для квиза на основе содержания этого ноутбука. Я сам выберу понравившиеся.

## Номер занятия

Возьми из имени файла ноутбука: `01`, `02`, `03`, `04` и т.д. (например, из `04_seminar_cnn.ipynb` → `04`).

## Требования к вопросам

1. Вопросы должны быть **несложными**, проверяющими базовое понимание ключевых концепций занятия.
2. Ответ — **максимально короткий**: не больше 5 слов.
3. Ответ **однозначный**, чтобы его мог проверить LLM as a judge.
4. Если есть несколько правильных вариантов ответа — включи их **все** в скобках или через «или».
5. **Не задавай вопросы по темам из раздела Recap** (Dropout, Optimizers, SGD vs Adam, общая регуляризация и т.п.) — только по основному материалу семинара (например, для CNN: свёртки, pooling, padding/stride, BatchNorm в контексте CNN, рецептивное поле, ResNet, работа с изображениями и т.д.).
6. Избегай вопросов, требующих развёрнутых объяснений или субъективных суждений.

## Формат ответа

Выдай результат в виде **одного кода блока**:

```
{номер_занятия}_{краткое_описание_темы_латиницей}
Текст вопроса?
Краткий ответ (альтернативные термины, синонимы).

{номер_занятия}_{другая_тема}
Текст вопроса?
Краткий ответ.
```

И так для всех 10 вопросов.

## Примеры хороших вопросов

- Какой параметр свертки задаёт ширину «рамки» вокруг изображения, чтобы сохранить размер карты?
  - **Ответ:** padding (отступ).
- Есть ли у слоя MaxPool2d обучаемые параметры?
  - **Ответ:** Нет (нет параметров).
- Для ядра 5×5 и stride=1 какое значение padding сохраняет пространственный размер?
  - **Ответ:** 2 (padding=2).

## Примеры плохих вопросов

- Объясните, как работает batch normalization (слишком развёрнутый ответ).
- Какие преимущества имеет Adam? (recap, субъективно.)
- Приведите примеры использования dropout (recap, много вариантов.)

## Порядок действий

1. Определи по имени файла номер занятия (`01`–`99`).
2. Просмотри ноутбук и выдели **ключевые темы основного материала** (исключая блок Recap).
3. Сформулируй 10 вопросов с короткими однозначными ответами.
4. Выведи их в указанном формате одним код-блоком.
