
# CPU vs GPU vs TPU

![](static/cpu_gpu.png)


#### CPU:

**Цель**: вычисления общего назначения

* Кол-во ядер: ~128
* Мощность ядер: 3.5 GHz

#### GPU:

**Цель**: узкоспециализированные вычисления, с большой степенью **распараллеливания**

* Кол-во ядер: ~7k
* Мощность ядер: ~1 GHz
