# Форматирование вопросов в семинаре

Проверь и отформатируй вопросы в ноутбуке семинара `$ARGUMENTS`.

## Форматирование вопросов

- Каждый вопрос оформи как **отдельная ячейка** с заголовком уровня 4.
- Формат строго: `#### ❓ **Вопрос**: текст вопроса?`
- Замени варианты вроде `**Вопрос:**`, `**Вопрос!**`, `### **Вопрос!**` на этот формат.
- Все вопросы должны содержать ответ, скрытый под спойлером:
```md
#### ❓ **Вопрос**: текст вопроса?

<details>

<summary><strong>Ответ</strong></summary>

Текст ответа</br>
Следующая строчка ответа</br>

</details>
```

Редактируй markdown-ячейки через `Edit` или `NotebookEdit`. Не меняй код в ячейках.
